Global variables:

Global variables can be used to replace certain 'tags' in your content by an easily editable value.
You could for example add a variable "phonenumber", and put your phone number in the value field.
If you then use "[phonenumber]" anywhere in your articles, your actual phone number will show up.
Changing the number on all articles then becomes as easy as changing the value for the variable on this page.
